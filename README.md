# MSX0_LCD

MSX0にi2c接続で文字を表示するサンプル。

C初心者のど素人につき、フツーこんな事しないだろってトコが散見されると思いますが、そこは自身で修正してください。

おそらくLCDの初期化方法などが異なるのだと思われますが、まだよくわかっていません。
